# Proyecto de final de grado 2ºK DAM
## Planteamiento del proyecto

Nuestro proyecto consiste en el desarrollo de una aplicación multiplataforma que permita gestionar de manera integral una empresa compuesta por varios centros veterinarios y de adopción, optimizando la administración de clientes, animales, citas, tratamientos, empleados y adopciones.  

**El sistema estará formado por dos aplicaciones principales:**

1. **Aplicación Web/Escritorio (para centros y empleados):**

    Diseñada para la gestión administrativa y operativa de los centros veterinarios y de adopción.

    Permitirá registrar empleados, clientes, animales, tratamientos, citas y adopciones, además de controlar la actividad diaria de cada centro.

2. **Aplicación Móvil (para clientes):**

    Orientada al cliente final.

    Permitirá consultar y gestionar información personal, ver el historial médico de sus mascotas, consultar y pedir citas, recibir recordatorios, acceder a información sobre tratamientos y explorar animales disponibles para adopción.
